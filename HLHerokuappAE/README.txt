General:

Web-UI Automation:
The tests for https://the-internet.herokuapp.com is written using Cucumber feature files, step definitions files and page model files.
The tests can be run from IntelliJ by right clicking on the feature(to run all scenario's) or an individual scenario

API Automation:
All tests for https://reqres.in/api is written in the a single class file APItest
The test can be run from IntelliJ by right clicking on the class file and Run.Also,select VM options and add -Dtestng.dtd.http=true
